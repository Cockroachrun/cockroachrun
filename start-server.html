Cockroach Run - Testing Instructions

Note: To fully test the 3D model functionality and other features, you need to serve these files from a web server.

Option 1: Testing with Python (easiest if installed)

If you have Python installed, open Command Prompt or PowerShell and run:

cd C:\Users\sk84l\OneDrive\Desktop\Cockroach-run-new
python -m http.server 8000

Then open http://localhost:8000 in your browser.

Option 2: Testing with XAMPP

If you have XAMPP installed:

  1. Copy this folder to C:\xampp\htdocs\cockroach-run
  2. Start Apache server from XAMPP Control Panel
  3. Open http://localhost/cockroach-run

Option 3: Visual Studio Code Live Server

If you have VS Code:

  1. Install "Live Server" extension
  2. Open this folder in VS Code
  3. Click "Go Live" button at the bottom right

Option 4: Simple Opening (limited functionality)

You can directly open the index.html file in your browser, but 3D models and some features may not work:

Tip: Make sure all your image and model files are in the correct locations:

assets/images/characters/German Cockroach with bg.png
assets/images/characters/American Cockroach with bg.png
assets/images/characters/Oriental Cockroach with bg.png
assets/models/German Cockroach.glb
assets/models/American Cockroach.glb
assets/models/Oriental cockroach.glb
